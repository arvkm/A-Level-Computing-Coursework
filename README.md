# A Level Computing Coursework - Catris!

## What is it?

This game is a version of Tetris themed around cats. It has unique features such as debuffs given to the player that block the "hold" button, speeding play up, and disallowing the player from rotating pieces!
It also features a cat that steals pieces from the matrix, disrupting play!
Note: gameplay can be buggy when pressing the hard drop button sometimes.

(Note: not affiliated with The Tetris Company. All art assets self-produced, sound assets are secondary source)
